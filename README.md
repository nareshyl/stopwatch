# 🍅 POMO — Focus Timer

A brutalist-aesthetic Pomodoro timer to supercharge your productivity.

## Features
- Adjustable work & break durations (via sliders)
- Progress bar tracking within each session
- Session cycle counter (4 sessions per cycle)
- Start / Pause / Reset / Skip controls
- Live browser tab title with countdown

## How to Use
Open `index.html` in any modern browser. No build step required.

## Tech Stack
- Pure HTML, CSS, JavaScript
- Google Fonts: Bebas Neue + Space Mono
- Web Crypto API not used here — standard setInterval timer
